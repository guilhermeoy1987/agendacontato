# Projeto Agenda de Contatos

Este repositório contém uma aplicação prática de agenda, desenvolvida para gerenciar nomes e telefones de forma dinâmica em uma interface web.

## O que foi realizado:
* **Inserção Dinâmica:** Adição de contatos na tabela sem a necessidade de atualizar a página.
* **Validação de Dados:** Lógica para evitar o cadastro de nomes ou números duplicados.
* **Manipulação de DOM:** Uso de JavaScript para criar e organizar elementos HTML em tempo real.
* **Interface Responsiva:** Estilização com CSS para garantir uma boa visualização em diferentes dispositivos.

## Tecnologias:
* HTML5
* CSS3
* JavaScript
* Git / GitHub
